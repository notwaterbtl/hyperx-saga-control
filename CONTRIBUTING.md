# Contributing

Thanks for your interest in HyperX Saga Control.

Useful contributions include:

- testing on Fedora, Bazzite, KDE Plasma, GNOME, and other Linux environments
- UI improvements
- packaging work
- protocol notes
- safe capture analysis
- additional HyperX device support

## Please do not submit

- proprietary HyperX/HP software
- HyperX NGENUITY binaries
- firmware dumps
- copyrighted assets
- raw captures containing unrelated private USB device traffic

If you submit USB captures, please keep them minimal and document exactly what setting was changed.

```bash
./run.sh
```

## Installed run

```bash
./install.sh
hyperx-saga-control
```
